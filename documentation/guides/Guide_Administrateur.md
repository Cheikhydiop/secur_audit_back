# Guide Administrateur — SmartAudit DG-SECU/Sonatel (SONATEL)

Bienvenue dans le guide de l'administrateur système pour la plateforme **SmartAudit DG-SECU/Sonatel**. Vos privilèges vous permettent de superviser l'ensemble de l'activité sur les sites Sonatel.

---

## 🛠️ 1. Gestion des Utilisateurs
- **Profils** : Super-Admin, Admin, Dirigeant, Inspecteur.
- **Ajout** : Créez des comptes pour les nouveaux agents.
- **Réinitialisation** : Gérez les mots de passe et changez les rôles.
- **Invitation** : Suivez les statuts d'acceptation des invitations.

---

## 🏗️ 2. Gestion des Sites
- **Annuaire des sites** : Liste complète avec codes, types (BT, technique, caisse) et zones.
- **Ajout dynamique** : Si un nouveau site est créé, ajoutez-le avec ses coordonnées GPS.
- **Historique** : Voir toutes les inspections passées pour un site spécifique.

---

## 🗓️ 3. Planning et Roadmaps
- **Planification** : Assignez une mission (site + date + inspecteur).
- **Import Roadmap** : Importez un planning annuel depuis un fichier Excel (compatible avec le format SONATEL DG/SECU).
- **Reconduction annuelle** : Un bouton permet de reconduire automatiquement le planning de l'année précédente vers la nouvelle année.

---

## 📊 4. Supervision des Rapports
- **Approbation** : Consultez les inspections soumises par les agents.
- **Audit Logs** : Accédez au journal complet des actions système pour la traçabilité.
- **Plans d'Actions** : Supervisez l'exécution des corrections sur tout le Sénégal.

---

## ⚙️ 5. Configuration Technique
- **Export Dashboard** : Générez des rapports agrégés de performance mensuelle.
- **Templates de questions** : Mise à jour du référentiel des points de contrôle (Score total : 73 points).
- **Notifications** : Configurez les alertes par email et in-app (ex: 7 jours avant l'échéance d'une action).

---

**Sécurité et Conformité**
Assurez-vous que tous les utilisateurs utilisent des mots de passe robustes. Les accès sont tracés dans le menu "Logs d'audit".
